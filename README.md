# PhysSim

Modular C++ physics simulation engine.  
Pet project to explore numerical methods and simulation techniques.

## Planned scenarios
- [ ] Billiards — rigid body collisions, Verlet integration
- [ ] Solar System — N-body gravity, Leapfrog integrator  
- [ ] Fluid — SPH particle simulation
- [ ] Atom model — quantum orbital visualization
